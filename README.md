## To install skins

drop `.xzp` file(s) into `Aurora/skins` folder

## To install coverflows

drop `.cfljson` file(s) into `Aurora/media/layouts`

## To install filters

drop `.lua` file(s) into `Aurora/User/Scripts/Content/Filters`
